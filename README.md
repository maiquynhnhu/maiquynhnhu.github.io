# be-nhu

Marketing portfolio website for **Mai Quynh Nhu** — showcasing campaign execution, KOL/KOC management, event coordination, and brand activation projects.

## What's inside

Single-page static site built with vanilla HTML/CSS/JS, including:

- **Hero** — intro, role description, and CTAs
- **Impact Stats** — animated counters for reach, interactions, content volume, events, sponsorship growth
- **Experience** — ACFC Vietnam (Marketing Collaborator), SHAT JSC (Marketing Assistant Intern)
- **Social Designs** — embedded Canva content planning board
- **Key Projects** — brand activations, store launches, university competitions, community events
- **Skills** — operations & docs, events & campaigns, data & analytics, content & tools
- **Education** — Van Lang University, Digital Marketing (2021–2025)
- **Contact** — email, LinkedIn, location

## Deploying

This is a static site. A working HTML file deployed at `https://hoangtnm.github.io` (see [index.html](index.html) for the source).
